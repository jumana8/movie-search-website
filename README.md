# movie-search-website
a simple web project that allows users to search for a movie & view its information 
Features
1- Search for movies by name
  2- Display movie poster
  3- Show movie rating
  4- Show movie information (title, year, etc.)
.
Technologies Used:
 HTML
,CSS
,JavaScript
,Movie API
.
Project Purpose
This project was created as a practice project to improve web development skills and working with APIs.
